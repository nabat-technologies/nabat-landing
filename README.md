## 📌 Overview

NABAT is an early-stage Layer 1 blockchain project focused on modular architecture, scalability, and Proof-of-Stake infrastructure.

The project is currently in active development and serves as a public build-in-progress of a next-generation blockchain system.

## 🧠 Project Vision

NABAT aims to explore and develop a modular blockchain architecture designed to improve:

- Scalability limitations in monolithic chains
- Execution flexibility for decentralized applications
- Separation of core blockchain layers (consensus, execution, data availability)

The goal is to build a more adaptable foundation for future Web3 systems.
